# mymall
mall商城购物系统学习

始于2019-10-17

## 实现功能

- 商品品牌管理

1. 添加品牌
2. 删除品牌
3. 更新品牌
4. 查询品牌
5. 获取品牌列表

- 会员登录注册管理

1. 注册
2. 登录
3. 获取用户权限

- 后台用户管理

1. 获取验证码
2. 检验验证码正确性

- 商品属性分类管理

1. 添加商品属性分类
2. 删除商品属性分类
3. 查询商品属性分类
4. 更新商品属性分类
5. 获取商品属性分类及其属性

## 引用技术

- SpringBoot
- Mybatis
- Swagger-UI
- Redis
- SpringSecurity
- JWT
